## Ограничения (вне MVP)

- **PDF-экспорт** — не реализован. Требует MS Word / LibreOffice на сервере.
- **Реестр приложений (Power Query)** — в оригинале автозаполнялся из Excel; в веб-версии вводится вручную.
- **Копирование вложенных файлов** (схемы, протоколы, фото) — не реализовано.
- **Персистентность** — форма не сохраняет данные между сессиями браузера.
- **АОСР3 / АОСР_старый / АООК_старый** — работают (аналогичная структура), отдельно не тестировались.
